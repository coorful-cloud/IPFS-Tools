# IPFS-Tools
A visual management tool for IPFS

# Usage method
1.Double click IPFSTools Setup 1.0.0.exe to install.
2.Configure nodes. Open a node at will.
3.Go to my files tab,view uploaded files, or upload pictures and metadata.
